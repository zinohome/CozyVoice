
## Local LiveKit client JS

Due to regional CDN instability, place a local copy of `livekit-client.umd.min.js` next to this README:

```bash
curl -o livekit-client.umd.min.js \
  https://cdn.jsdelivr.net/npm/livekit-client@2/dist/livekit-client.umd.min.js
```

Or keep the script tag pointing to the CDN if your network reliably reaches `unpkg.com` / `jsdelivr`.
